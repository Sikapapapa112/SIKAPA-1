✅ Setup Steps

1 Create Firebase Project
Go to console.firebase.google.com and create a new project
2 Enable Authentication
Enable Email/Password authentication in Firebase Console
3 Create Realtime Database
Create a Realtime Database in test mode
4 Update firebase-config.js
Copy your Firebase config values from Firebase Console
5 Run Local Server
Run: python -m http.server 8000 in the SIKAPA folder
6 Open in Browser
Go to http://localhost:8000 in your browser
7 Create Admin Account
Sign up with the ADMIN_EMAIL from firebase-config.js
